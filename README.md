# VoiceCorpusProject
A corpus created from 384 voices of both males and females with a Central Mexican accent.

Este repositorio almacena distintos audios organizados por género y rangos de edad. Cada rango de edad tiene 64 audios.

## Fuentes de los Audios

### Common Voice
Una parte de los audios en este repositorio proviene del proyecto [Common Voice de Mozilla](https://commonvoice.mozilla.org/). Se tomaron audios de distintas personas con una duración mayor a 5 segundos.

### Audios de YouTube
Se muestra el nombre de la persona, la cual se verificó que contara con acento del centro de México, después aparece su edad en el momento en que grabó el video de Youtube, posteriormente se encuentra el link de Youtube y finalmente el intervalo del video que se tomó para generar la muestra de audio.
